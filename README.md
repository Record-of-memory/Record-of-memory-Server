# 우리기억

<br />
<br />

## 기획 배경

**😉 우리가 발견한 불편한 점** <br />
”친구들이랑 보냈던 시간을 기록하고 싶어” <br />
”마음이 맞는 내 덕질메이트랑 덕질일기 써보고 싶어” <br />
”예전에 교환일기 썼던 추억을 되살려보고 싶어” <br />
”뉴스, 시사, 정치처럼 민감한 주제지만 내 생각을 공유하고 싶어” <br />
“올해 친구랑 함께 했던 일상, 여행 등의 추억들을 한번에 예쁘게 보고 싶어” <br />
<br/>
**💡 그래서 이렇게 해결하고자 해요** <br />
다른 사람은 볼 수 없는 우리만의 공유 일기장이 필요해! <br />
동시에 혼자 쓰는 다이어리도 같이 있었으면 좋겠어! <br />
SNS는 다른 사람을 의식하게 되어서 불편해, 정말 쓰고 싶은 내 얘기를 자유롭게 하고 싶어! <br />
친구랑 언제 뭘 했는지 편하게 한번에 보고 싶어! <br />

**⇒ 우리끼리 편하게 쓸 비밀 다이어리가 있다면 어떨까?**
